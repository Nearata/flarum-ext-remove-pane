# Remove Pane

> A [Flarum](https://flarum.org) extension that remove the discussions side-pane.

## Installation

```sh
composer require nearata/flarum-ext-remove-pane
```

## Updating

```sh
composer update nearata/flarum-ext-remove-pane
php flarum cache:clear
```
